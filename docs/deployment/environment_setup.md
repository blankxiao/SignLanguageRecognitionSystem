# 环境配置说明

## 1. 基础环境
- Python 3.8+
- CUDA 11.x (可选)
- cuDNN 8.x (可选)

## 2. 依赖包
### 2.1 核心依赖
- PyTorch >= 1.8.0
- torchvision >= 0.9.0
- OpenCV >= 4.5.0
- NumPy >= 1.19.0
- Pillow >= 8.0.0

### 2.2 其他依赖
- matplotlib
- seaborn
- scikit-learn
- PyQt6
- mediapipe

## 3. 安装步骤
1. 创建虚拟环境
2. 安装PyTorch
3. 安装其他依赖
4. 验证安装

## 4. 常见问题
- CUDA版本兼容性
- 依赖包冲突解决
- 环境变量配置
